 Inventory Management System

Overview

The Inventory Management System is a software application designed to help businesses manage their inventory efficiently. It allows users to maintain product details, track stock levels, manage suppliers, record purchases and sales, and monitor inventory in real time.

Features

- Add, update, and delete products
- Manage product categories
- Manage supplier information
- Track inventory stock
- Record purchase transactions
- Record sales transactions
- View available stock
- Simple and user-friendly interface

Technologies Used

- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL

How to Run

1. Download or clone the repository.
2. Import the SQL database into MySQL.
3. Configure the database connection.
4. Start the Apache and MySQL services using XAMPP.
5. Open the project in your web browser.
